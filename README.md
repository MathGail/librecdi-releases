# LibreCDI — Releases

Ce dépôt contient les releases officielles de **LibreCDI**, logiciel de gestion de bibliothèque scolaire (CDI).

## À propos

LibreCDI est un logiciel de gestion de CDI (Centre de Documentation et d'Information) destiné aux établissements scolaires. Il permet de gérer le catalogue de livres, les emprunts, les élèves et les codes-barres.

## Téléchargement

Rendez-vous dans l'onglet [Releases](../../releases) pour télécharger la dernière version.

## Installation

1. Téléchargez le fichier `LibreCDI-Setup.exe` depuis la dernière release
2. Lancez l'installeur et suivez les instructions
3. Consultez le manuel d'utilisation inclus dans le package
